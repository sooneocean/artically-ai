# 產業別模板差異化設計

## 電商 / 品牌
- 重點：Hero 視覺、產品卡片、CTA、信任徽章
- 報告加強：色彩飽和度、視覺節奏、商品特寫比例

## SaaS / 軟體
- 重點：功能區塊、Pricing、轉換 CTA
- 報告加強：資訊層級、對比/可讀性、價目清晰度

## 內容媒體 / 社群
- 重點：內容模組、互動 CTA、社群導流
- 報告加強：版面密度、圖文比例、互動誘因

## 教育 / 課程
- 重點：課程結構、信任背書、CTA 位置
- 報告加強：學習路徑、權威背書區塊

## 金融 / 專業服務
- 重點：信任與穩定感、風險揭露
- 報告加強：視覺嚴謹度、資訊層級
