# 定價與點數制（建議）

## 一口價（買斷包）
- 單一品牌拆解包：NT$ 9,900 / US$ 299
- 多品牌（5 個內）：NT$ 29,900 / US$ 899

## 訂閱點數
- 基本方案：NT$ 990 / 月 → 100 點
- 成長方案：NT$ 2,990 / 月 → 400 點
- 企業方案：客製（點數池 + 報表模板）

## 點數消耗規則（範例）
- 官網拆解：10 點 / URL
- 社群拆解：8 點 / 平台
- 廣告拆解：12 點 / 活動
- 產出 PDF：5 點 / 份
- 啟用 ComfyUI 視覺補強：20 點 / 次

## 點數加值
- 額外購買：100 點 NT$ 600
